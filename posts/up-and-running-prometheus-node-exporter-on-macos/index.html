Up and Running Prometheus node_exporter on macOS

27 March 2023·1 min
  1. Install node_exporter
brew install node_exporter
  1. Start node_exporter
brew services start node_exporter
  1. Now you can see the logs at localhost:9100/metrics
open -n -a Safari localhost:9100/metrics
